REQUIREMENTS:

Python 3.8 or newer

Ollama running locally on port 11434

Packages: requests, rich

INSTALL STEPS:

Open a terminal inside the extracted 'albert' folder.

Run:
chmod +x install.sh
./install.sh

Restart your terminal or reload with:
source ~/.bashrc

Test by typing:
albert hello

IF YOU SEE AN ERROR:

Ensure ~/.local/bin is in your PATH:
echo $PATH
export PATH="$PATH:$HOME/.local/bin"

TO REMOVE:
rm ~/.local/bin/albert
and delete the alias line from ~/.bashrc
